# 💸 Transact — Finance Tracker Web App

A modern and minimal **finance tracking web app** built using **React + Tailwind CSS**.
Track your income, expenses, and manage transactions with a clean, smooth UI.

---

## 🚀 Live Demo

🔗 https://transact-your-transactions.netlify.app

---

## ✨ Features

* ➕ Add new transactions (Income / Expense)
* 🗑️ Delete transactions instantly
* 🔍 Search transactions in real-time
* 🎯 Filter by categories (Food, Shopping, Housing, etc.)
* 💰 Auto-calculated balance, income, and expenses
* 🎨 Modern UI with smooth animations & micro-interactions
* ⚡ Fast and responsive design

---

## 🧠 Tech Stack

* **React (Vite)**
* **Tailwind CSS**
* **Context API** (State Management)
* **Lucide Icons**

---

```
│── contexts/
│   └── TransactionContext.jsx
│
│── App.jsx
│── main.jsx
```

---

## ⚙️ How It Works

* Global state is managed using **React Context API**
* Transactions are stored in a centralized context
* Filtering and searching are computed dynamically
* UI updates instantly on add/delete actions
